# 📘 Assignment: REST APIs with FastAPI

## 🎯 Objective

Build a simple REST API using the FastAPI framework. Students will practice creating endpoints, handling request data, and returning structured JSON responses.

## 📝 Tasks

### 🛠️ Create API Endpoints

#### Description
Set up a FastAPI application with a few basic endpoints that demonstrate how REST APIs receive requests and return responses.

#### Requirements
Completed program should:

- Create a FastAPI app instance.
- Include a `GET` endpoint that returns a welcome or status message.
- Include a `GET` endpoint that returns a list of sample items as JSON.
- Run successfully as a local web server.


### 🛠️ Add Request Handling

#### Description
Expand the API by adding endpoints that accept input and return structured results. Focus on reading path parameters or request body data and producing clear JSON responses.

#### Requirements
Completed program should:

- Add at least one endpoint that accepts a path parameter.
- Add at least one endpoint that accepts data from the user.
- Return JSON responses with clear field names and values.
- Handle valid requests without errors.
- Include example requests or responses in code blocks if helpful while testing.
